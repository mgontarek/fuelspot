# FuelSpot

A web app for ultra-cyclists to find the nearest open resupply stop along their planned route.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Tests

```bash
npm test          # watch mode
npm run test:run  # single run
```

## Build

```bash
npm run build
npm run preview   # preview production build locally
```

## Deploy

Pushes to `main` trigger GitHub Actions which runs tests and deploys to GitHub Pages.
